# Neptun-LevelUP
Ez a projekt egy átfogóbb Userscript gyűjtemény, amelynek célja az új fajta Neptun tanulmányi rendszer gyakran lassú felületének modernizálása, biztonságosabbá tétele és funkcionális bővítése. A fejlesztés során a felhasználói élmény (UX) és a hatékonyság növelése volt a fő szempont.
